Modelli di framework web Astro per siti basati sui contenuti. Copre raccolte di contenuti con schemi e caricatori Zod, architettura dell'isola con direttive di idratazione selettiva, transizioni di visualizzazione con ClientRouter, modalità di rendering lato server e ibride, isole server, Astro DB con astro:db, middleware con onRequest e integrazioni framework (React, Svelte, Vue).

Da utilizzare durante la creazione di siti Web basati sui contenuti, la configurazione di strategie di idratazione dell'isola, l'impostazione di transizioni di visualizzazione, la scelta tra rendering statico e server, l'integrazione di componenti del framework dell'interfaccia utente, la definizione di schemi di raccolta di contenuti o l'aggiunta di middleware.
